# Changelog - SIDM (Sistema Integrado de Dados Municipais)

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato baseia-se em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0-beta] - 2026-06-21

### Adicionado
- **Portal Web Inicial**: Lançamento da interface Next.js 14+ com design em Glassmorphism e responsividade total.
- **Documentação da API**: Rota `/docs` contendo a tabela completa de queries e parâmetros de consulta aceitos pela API GraphQL do SIDM.
- **Seção Acadêmica**: Rota `/artigo/conic` criada para ler a publicação preliminar de fomento do projeto com estilização adaptada para artigos científicos.
- **Suporte Mobile**: Navegação adaptável com menu hambúrguer interativo para telas pequenas.

### Sobre a versão Beta
A versão `0.1.0-beta` sinaliza a fase de testes iniciais e lapidação do sistema SIDM antes da publicação da versão estável `1.0.0`. O esquema da API GraphQL pode sofrer quebras de contrato de dados durante esta fase.
